# assets/

Espaço para exemplos de criativos visuais já aprovados pela MAI. Quando
disponíveis, devem ser referenciados no PASSO 5 do `SKILL.md` para calibrar
o gerador de imagem.

## Convenção de nomes

```
{formato}-{cor}-{tema-curto}.png
```

Exemplos:
- `noticia-laranja-claude-cowork.png`
- `tutorial-azul-mcp-google-drive.png`
- `comparativo-roxo-claude-vs-gpt.png`
- `oportunidade-verde-gemini-trial.png`

## O que coloque aqui

- PNG ou JPG, 1080x1080
- Criativos que efetivamente foram publicados e tiveram boa resposta
- Idealmente 2-3 exemplos por formato

Quando esta pasta tiver exemplos, atualize o PASSO 5 do SKILL.md citando-os
explicitamente como referência visual.
